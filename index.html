Hosts & Setup

iOS Host: IOS

Steps to set up Growtopia on iOS using Surge 5:

[General]
bypass-system = true

[Rule]
FINAL,DIRECT

[Host]
growtopia1.com = 51.77.180.169
growtopia2.com = 51.77.180.169
www.growtopia1.com = 51.77.180.169
www.growtopia2.com = 51.77.180.169

Android Host: ANDROID

Steps to set up Growtopia on Android using Powertunnel:

1 - Download Powertunnel APK
2 - Click on "Powertunnel" button to copy host link
3 - Open Powertunnel → three-line icon → Host Settings
4 - Paste copied host URL
5 - Change host update period: 12 hours → On Start
6 - Log in to GTPS!

PC/Windows Host

Steps to set up Growtopia on Windows:

1 - Press ⊞ Win+R (Windows Key + R) to open the "Run" window.
2 - Type in C:\Windows\System32\drivers\etc
3 - Find the file named hosts
4 - Right-click "hosts" → Edit with Notepad or text editor
5 - Add the following IPs:

51.77.180.169 growtopia1.com
51.77.180.169 growtopia2.com
51.77.180.169 www.growtopia1.com
51.77.180.169 www.growtopia2.com

6 - Save the file and log in to Growtopia
