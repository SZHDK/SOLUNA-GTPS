iOS Setup (Surge 5)

Instructions

1 - Download "Surge 5" from the App Store.
2 - Press "OK".
3 - Tap "Default.conf".
4 - Click "IMPORT" → Download Profile from URL.
5 - Paste the iOS host link (copy from button below).
6 - Tap "OK" then "Done".
7 - Press "SETUP" → Agree to policy → Allow VPN configuration.
8 - Open Growtopia and connect!
      
[General]
bypass-system = true

[Rule]
FINAL,DIRECT

[Host]
growtopia1.com = 51.77.180.169
growtopia2.com = 51.77.180.169
www.growtopia1.com = 51.77.180.169
www.growtopia2.com = 51.77.180.169

Android Setup (Powertunnel)

Instructions

1 - Download Powertunnel APK
2 - Tap button to copy host URL
3 - Open Powertunnel → Host Settings
4 - Paste URL → Change update period: On Start
5 - Log in to GTPS!

PC / Windows Setup

Instructions

2 - Open "hosts" with Notepad
3 - Add these lines:
